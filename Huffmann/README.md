### Aufgabenstellung
Bitte hier ihre Huffman-Aufgabe abgeben. Dazu gehören:
- Binärer, beschrifteter Baum (Kann auch ein Printscreen sein)
- Codetabelle
- Ihr gewähltes Wort als binärer Huffmancode

### Tabelle
-  11 = **L**
-  1011 = **Leerzeichen**
-  1010 = **T**
-  1001 = **!**
-  1000 = **E**
-  011 = **H**
-  010 = **O**
-  001 = **W**
-  000 = **A**
